# Startup Project

This is a Django-based web application for simulation and startup management.

## Features
- Simulation canvas
- Agent management
- Demo and about pages
- Static and media file support

## Project Structure
- `core/` - Main app logic
- `simulation/` - Simulation features
- `myproject/` - Django project settings
- `static/` - Static files (JS, CSS, images)
- `templates/` - HTML templates

## Setup
1. Clone the repository:
   ```sh
   git clone https://github.com/arpit123de/startup.git
   ```
2. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
3. Run migrations:
   ```sh
   python manage.py migrate
   ```
4. Start the development server:
   ```sh
   python manage.py runserver
   ```
